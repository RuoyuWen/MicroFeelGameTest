【超快速上传 - 3步搞定】
================================================================

第一步：在 GitHub 创建仓库
  1. 访问 https://github.com/
  2. 点击右上角 "+" → "New repository"
  3. 填写仓库名：ai-rpg-test-system
  4. 选择 Public 或 Private
  5. 不要勾选 "Initialize with README"
  6. 点击 "Create repository"

第二步：上传代码
  打开项目文件夹，运行以下命令：
  
  git init
  git add .
  git commit -m "Initial commit: AI RPG 测试系统"
  git branch -M main
  git remote add origin https://github.com/你的用户名/ai-rpg-test-system.git
  git push -u origin main

第三步：完成！
  访问你的 GitHub 仓库查看

【不会用 Git？没关系！】
================================================================

方法 1：使用 GitHub Desktop（图形界面）
  1. 下载：https://desktop.github.com/
  2. 安装并登录
  3. File → Add Local Repository → 选择项目文件夹
  4. Commit → Publish to GitHub

方法 2：使用 VS Code
  1. 在 VS Code 打开项目
  2. 点击左侧 "Source Control" 图标
  3. "Initialize Repository"
  4. "Publish to GitHub"
