【如何开始测试】

⚠️ 重要：不能直接双击 HTML 文件！会有 CORS 错误。

正确方法：
  Windows: 双击 "启动服务器.bat"
  Mac/Linux: 运行 "python3 启动服务器.py"
  VS Code: 使用 "Live Server" 扩展

详细说明请看：CORS问题解决方案.md

1. 按上述方法启动服务器，浏览器访问 http://localhost:8000

2. 输入你的 OpenAI API Key
   - 如果没有，请访问：https://platform.openai.com/api-keys
   - 需要有 OpenAI 账号和可用余额

3. 选择模型（建议）：
   ⭐ GPT-4o-mini（推荐，性价比高）
   ⭐ GPT-4o（质量最佳）

4. 默认的 System Prompt 已经配置好，可以直接使用

5. 点击"开始"进入场景设置

【快速测试场景】

故事总结：你来到一个小镇，想在酒馆找到冒险伙伴。

NPC列表：老板娘玛丽、战士托尔、盗贼艾莉丝

NPC目标：老板娘想卖东西，战士想找队友，盗贼想加入队伍。

【测试流程】

第一步：配置页
├─ 输入 API Key
├─ 选择模型
├─ 配置三个模块（可用默认）
└─ 点击"开始"

第二步：场景初始化
├─ 输入故事背景
├─ 输入 NPC 列表
├─ 输入 NPC 目标
└─ 点击"开始对话"

第三步：对话互动
├─ 输入玩家对话
├─ 观察 NPC 回应（名字+内容+情绪）
├─ 继续对话
└─ 点击"结束对话"

第四步：总结与下一幕
├─ 自动生成场景总结
├─ 自动生成下一幕故事
├─ 点击"下一幕"继续
└─ 或点击"返回配置"重新开始

【常见问题】

Q: API 调用失败？
A: 检查 Key 是否正确，账户是否有余额

Q: NPC 回应格式混乱？
A: 尝试启用 JSON Mode

Q: 响应很慢？
A: 正常现象，AI 需要思考时间（5-15秒）

Q: 想重新开始？
A: 点击"返回配置"或刷新页面
