⚠️⚠️⚠️ 重要提示 ⚠️⚠️⚠️

不能直接双击 index.html 文件！
会出现 CORS 跨域错误，无法调用 OpenAI API。

================================================================

【最简单的启动方法】

Windows 用户：
  1. 双击 "启动服务器.bat" 文件
  2. 等待浏览器自动打开
  3. 开始使用！

Mac/Linux 用户：
  1. 在终端运行：python3 proxy_server.py
  2. 浏览器会自动打开
  3. 开始使用！

【重要】使用 proxy_server.py 启动，内置 OpenAI API 代理！

【新功能】AI 现在有记忆了！
  - NPC 会记住之前的对话（最近10轮）
  - 可以引用之前说过的话
  - 对话更连贯、更真实！

================================================================

【如果启动服务器失败】

1. 检查是否安装了 Python
   下载地址：https://www.python.org/downloads/

2. 或使用 Node.js：
   命令：npx http-server -p 8000

3. 查看详细解决方案：
   打开 "CORS问题解决方案.md" 文件

================================================================

【启动成功的标志】

✓ 浏览器地址栏显示：http://localhost:8000
✗ 如果显示 file:///... 则启动方式不对

================================================================

【接下来】

1. 输入 OpenAI API Key
2. 选择模型（推荐 GPT-4o-mini）
3. 点击"开始"
4. 查看 "快速开始.md" 了解详细使用方法
